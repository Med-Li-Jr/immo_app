# :computer: Property Rent Management

A property rental web app that allows a user to manage its real estate assets and also to search for properties to rent with specific criteria.

## Technologies used

|-- <a href="https://angular.io/" target="_blank">Angular</a> &nbsp; |-- <a href="https://docs.microsoft.com/en-us/aspnet/core/" target="_blank">.Net Core</a> &nbsp; |-- <a href="https://www.microsoft.com/en-us/sql-server/" target="_blank">Sql Server</a> &nbsp; |-- <a href="https://www.mapbox.com/" target="_blank">MapBox API</a>
